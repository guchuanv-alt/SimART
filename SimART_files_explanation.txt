一、项目总体用途
----------------
SimART 是一个基于 ROS1 的无线通信与感知仿真平台。它把 C++/Qt/VTK 图形界面、Sionna RT/SYS 射线追踪和链路仿真、AirSim/Unreal Engine 实时可视化、rosbag 录制/回放/重仿真、CKM(Channel Knowledge Map) 生成等功能串在一起。

核心流程大致是：
1. 载入 3D 场景或样例地图。
2. 配置基站、坐标变换、Sionna 场景 XML、ROS 输入 topic。
3. 从 UAV 位姿 topic 或 rosbag 中获取轨迹。
4. 用 Sionna 计算 RF 路径、SYS 链路指标、beam/codebook 结果。
5. 在 GUI、UE live view、ROS topic、CSV/rosbag 中查看或保存结果。

三、ROS 消息包
--------------

3.1 rf_msgs/
~~~~~~~~~~~~
rf_msgs 是 RF 射线追踪观测消息包，用于描述接收机、基站和传播路径。

rf_msgs/package.xml
  ROS catkin 包元数据。声明 message_generation、message_runtime、std_msgs、geometry_msgs 等依赖。

rf_msgs/CMakeLists.txt
  catkin 构建文件。注册 RfPathObservation.msg、RfAnchorObservation.msg、RfObservationArray.msg 并生成消息。

rf_msgs/msg/RfPathObservation.msg
  单条传播路径观测。包含路径类型、阶数、路径 ID、幅度/增益/功率、时延、Doppler、AoA、TX/RX 坐标以及路径折返点。

rf_msgs/msg/RfAnchorObservation.msg
  单个 anchor/base station 的 RF 观测。包含 anchor ID、名称、位置和多条路径。

rf_msgs/msg/RfObservationArray.msg
  一帧 RF 观测集合。包含 header、接收机位置/速度以及多个 anchor 的路径观测。


3.2 sionna_sys_msgs/
~~~~~~~~~~~~~~~~~~~~
sionna_sys_msgs 是 Sionna SYS 链路层/系统级指标消息包。

sionna_sys_msgs/package.xml
  ROS 包元数据。声明 std_msgs、geometry_msgs、message_generation/message_runtime。

sionna_sys_msgs/CMakeLists.txt
  注册 SysConfig、SysServingLink、SysCandidateObservation、SysObservationArray 消息并生成。

sionna_sys_msgs/msg/SysConfig.msg
  SYS 仿真配置。包含是否启用、子载波数、子载波间隔、OFDM 符号数、温度、BLER 目标、MCS 表、发射功率、载频等。

sionna_sys_msgs/msg/SysServingLink.msg
  当前服务链路摘要。包含服务基站、速率、有效 SINR、MCS、解码 bit、ACK/NACK 状态、频谱效率等。

sionna_sys_msgs/msg/SysCandidateObservation.msg
  每个候选基站的 SYS 观测。包含路径数量、最强路径、候选速率、SINR、MCS、传输块状态、BLER 目标等。

sionna_sys_msgs/msg/SysObservationArray.msg
  一帧 SYS 观测集合。包含接收机状态、调度策略、候选列表、配置和服务链路。


3.3 sionna_beam_msgs/
~~~~~~~~~~~~~~~~~~~~~
sionna_beam_msgs 是 beamforming、码本和波束选择结果消息包。

sionna_beam_msgs/package.xml
  ROS 包元数据。描述为 Sionna beam codebook 和 per-BS beam observation 的消息定义。

sionna_beam_msgs/CMakeLists.txt
  注册 BeamEntry、BeamCodebook、BeamObservation、BeamObservationArray 消息并生成。

sionna_beam_msgs/msg/BeamEntry.msg
  一个 beam 的条目。包含 beam index、二维 beam 网格索引、复数权重的实部/虚部数组。

sionna_beam_msgs/msg/BeamCodebook.msg
  beam codebook 定义。包含 codebook ID/type、阵列尺寸、过采样、归一化设置、beam 数量和 beam 列表。

sionna_beam_msgs/msg/BeamObservation.msg
  单个基站的一帧 beam 观测。包含选择模式、codebook、手动/预测/oracle beam、增益、top-k、特征向量、是否命中等。

sionna_beam_msgs/msg/BeamObservationArray.msg
  多个基站 beam 观测的集合。包含仿真序号、odom 时间戳、frame_id、codebook_id 和 observations。


四、SimART_GUI ROS/Qt GUI 包
---------------------------
SimART_GUI 实际 ROS 包名为 airsim_gui_UErealtime，是主图形界面和运行时工具所在目录。

SimART_GUI/package.xml
  GUI 包元数据。声明 Qt、VTK、assimp、roscpp、tf/tf2、geometry/nav/std/sensor msgs、rf_msgs、sionna_sys_msgs、sionna_beam_msgs 等依赖。

SimART_GUI/CMakeLists.txt
  GUI 包构建文件。启用 Qt AUTOMOC/AUTORCC/AUTOUIC，查找 Qt5、VTK、assimp、catkin；构建主 GUI 可执行文件；可选启用 AirSim C++ RPC live view。

SimART_GUI/resources.qrc
  Qt 资源文件。当前把 qml/UeLiveView.qml 打包进资源系统。

~~~~~~~~~~~
SimART_GUI/launch/sionna_sim_only_topic.launch
  启动 Sionna 在线仿真节点 sionna_sim_only_topic2.py。配置样例地图路径、Sionna XML、基站 JSON、ROS-to-map 变换、odom topic、仿真频率、RF/SYS 输出 topic、载频、Mitsuba variant、ray tracing 深度、预览渲染参数等。

八、文件之间的关键关系
----------------------
1. GUI 主程序从 SimART_GUI/src/main.cpp 进入，主要逻辑在 MainWindow.cpp。
2. MainWindow 通过 RosBridge 订阅/发布 ROS 数据，通过 Scene3DWidget 展示 3D 场景，通过 AirSimViewController 展示 UE/AirSim live view。
3. Sionna 仿真主要由 sionna_sim_only_topic2.py、sionna_internal_socket_bridge.py 和 sionna_resim_socket_worker.py 完成。
4. RF/SYS/beam 结果通过 rf_msgs、sionna_sys_msgs、sionna_beam_msgs 三组 ROS 消息发布。
5. 样例地图中的 *.xml + meshes/*.ply 是 Sionna RT 使用的传播场景；*.glb/*.fbx/*.blend 更多用于可视化、地图编辑或 UE/AirSim 导入。
6. config/ROS_to_3D_matrix.json、airsim_to_3D_matrix.json 等矩阵文件把 ROS、3D scene、AirSim 坐标系统一起来。
7. Tutorials 目录说明如何创建地图和使用 GUI；images 目录提供 README/教程中的截图和动图。
